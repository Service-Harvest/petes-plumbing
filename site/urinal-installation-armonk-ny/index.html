The installation process

We start by confirming the wall structure can support the fixture and that the existing rough-in plumbing lines up with the unit you're installing, or adjusting it if it doesn't. From there, we set the fixture, connect the water supply and flush valve, and tie in the drain line.

Once everything is connected, we test the flush and drain thoroughly and check for leaks around every connection point before we consider the job finished.

What affects the cost

Installing a urinal where plumbing already exists for one is a simpler job than adding a completely new fixture location, which requires new supply and drain lines. The type of flush valve — manual or sensor-operated — also affects the scope of the work.

We'll assess your existing restroom plumbing and give you a clear price before starting anything.

Restroom upgrades for local businesses

We see this request most often from small offices, shops, and other commercial properties around Armonk that are updating restrooms to meet current code or simply modernizing an older space. Commercial restroom fixtures have specific code and water-efficiency standards that differ from a typical home installation.

The EPA's WaterSense program has useful guidance on the EPA's WaterSense program on water-efficient fixtures if you're weighing options for a restroom update — we can help you translate that into the right fixture for your space.

Keeping it running smoothly

Most urinal problems come down to a worn flush valve, a partial clog, or a drain line that wasn't sized correctly during installation. Getting the install right the first time avoids most of these issues down the line.

If you already have restroom fixtures that need attention, or you're planning a broader update, it's worth asking about our broader commercial plumbing support while we're on site.

When to call a pro

Call us if you're adding a urinal to an existing restroom, updating fixtures as part of a renovation, or dealing with a urinal that's leaking, clogged, or not flushing properly. Commercial restroom work should always go through a licensed plumber, both for code reasons and for the practical fact that a bad restroom fixture affects your customers or employees, not just you.

If your property also needs help staying compliant with backflow testing or maintaining a restaurant's grease trap, we handle those alongside restroom work too.

Frequently asked questions

Do you install urinals in homes as well as businesses?

Most of our urinal installations are for commercial or light commercial properties, but we can discuss a residential installation as well.

Can you add a urinal where there wasn't one before?

Yes, though it requires running new supply and drain lines to the fixture location, which adds to the scope of the job.

Do you install sensor-operated flush valves?

Yes, we install both manual and sensor-operated flush valves, depending on what you prefer for your restroom.

How long does a urinal installation take?

It depends on whether existing plumbing is already in place. We'll give you a specific timeline after assessing your restroom.

Do you handle multiple fixture installs in one visit?

Yes, if you're updating an entire restroom, we can install multiple fixtures during the same visit.

Will a new urinal meet current water-efficiency code?

We install fixtures that meet current water-efficiency standards and can point you toward options that qualify for any applicable requirements.
